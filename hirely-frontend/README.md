## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm
- Python 3.12+
- PostgreSQL database (or use Neon cloud)
- Google Generative AI API key
- VAPI API credentials

### Installation

#### 1. Clone the Repository
```bash
git clone https://github.com/red445992/Hirely---Ai-powered-Job-portal.git
cd Hirely---Ai-powered-Job-portal
```

#### 2. Frontend Setup
```bash
cd hirely-frontend
npm install

# Create .env file
# Add the following variables:
# DATABASE_URL="your_postgresql_connection_string"
# NEXT_PUBLIC_API_URL="http://localhost:8000"
# GOOGLE_GENERATIVE_API_KEY="your_google_ai_key"
# NEXT_PUBLIC_VAPI_WEB_TOKEN="your_vapi_token"
# NEXT_PUBLIC_VAPI_WORKFLOW_ID="your_vapi_workflow_id"

# Initialize Prisma
npx prisma generate
npx prisma db push

# Run development server
npm run dev
```

Frontend will be available at `http://localhost:3000`

#### 3. Backend Setup
```bash
cd backend
pip install -r requirements.txt

# Create .env file in backend directory
# Add database credentials and API keys

# Run migrations
python manage.py migrate

# Create superuser (optional)
python manage.py createsuperuser

# Run development server
python manage.py runserver
```

Backend API will be available at `http://localhost:8000`
